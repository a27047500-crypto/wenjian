# 小范围轻量服务器上线方案

适合场景：

- 公司内部 5 到 30 人使用
- 想尽快上线
- 不想自己维护办公室电脑常开
- 想直接发一个网址给同事

## 推荐方案

推荐购买：

- 腾讯云轻量应用服务器
- 地域：香港
- 系统：Ubuntu 24.04 LTS
- 配置：2 核 4G
- 磁盘：50GB 以上

原因：

- 购买和管理都比传统云服务器简单
- 小范围试用足够
- 香港地域通常不用做中国内地 ICP 备案
- 同事不在同一个办公室也可以访问

## 购买步骤

### 第 1 步：注册腾讯云

1. 注册腾讯云账号
2. 完成实名认证
3. 打开轻量应用服务器购买页

### 第 2 步：选择配置

按下面选：

1. 地域：香港
2. 镜像：Ubuntu 24.04 LTS
3. 套餐：2 核 4G，50GB 以上
4. 带宽：默认即可，后面不够再升
5. 登录方式：密码
6. 设置 root 密码并保存好

### 第 3 步：放通端口

购买完成后，在防火墙或安全组里放通：

- `22`
- `80`
- `443`
- `3100`

如果后面配置了 Nginx 和 HTTPS，最终给同事访问时主要用：

- `80`
- `443`

## 部署步骤

### 第 1 步：登录服务器

在你本机 PowerShell 执行：

```powershell
ssh root@你的服务器公网IP
```

### 第 2 步：上传项目

把本地这个目录上传到服务器：

- [sop-company-app](</C:/Users/Administrator/Desktop/word/sop-company-app>)

推荐上传到：

```bash
/root/upload/sop-company-app
```

### 第 3 步：安装运行环境

项目里现成脚本：

- [deploy/linux/setup-ubuntu.sh](</C:/Users/Administrator/Desktop/word/sop-company-app/deploy/linux/setup-ubuntu.sh>)

执行：

```bash
cd /root/upload/sop-company-app
chmod +x deploy/linux/setup-ubuntu.sh
sudo bash deploy/linux/setup-ubuntu.sh
```

### 第 4 步：正式安装

项目里现成脚本：

- [deploy/linux/install.sh](</C:/Users/Administrator/Desktop/word/sop-company-app/deploy/linux/install.sh>)

执行：

```bash
cd /root/upload/sop-company-app
chmod +x deploy/linux/install.sh
sudo APP_SLUG=sop-company-app \
APP_USER=flowdocs \
APP_DIR=/opt/sop-company-app \
DATA_ROOT=/var/lib/sop-company-app/data \
PORT=3100 \
HOST=127.0.0.1 \
DOMAIN=_ \
bash deploy/linux/install.sh
```

### 第 5 步：检查服务

```bash
systemctl status sop-company-app.service --no-pager
curl http://127.0.0.1:3100/api/health
nginx -t
```

## 先不绑域名时怎么给同事用

如果你暂时不买域名，先直接发公网 IP：

```text
http://你的服务器公网IP
```

因为安装脚本已经把 Nginx 代理到应用了，所以你给同事的不需要带 `:3100`。

## 如果后面想换正式域名

做法：

1. 买一个域名
2. 添加 A 记录指向服务器公网 IP
3. 修改 Nginx 的域名配置
4. 用 Certbot 配 HTTPS
