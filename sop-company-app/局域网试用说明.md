# 局域网试用

## 最简单的方法

1. 选一台办公室电脑作为临时主机
2. 在这台电脑上双击 `局域网启动SOP公司应用.bat`
3. 记下弹出的局域网地址，例如：

```text
http://192.168.1.23:3100
```

4. 让同事在同一办公室网络下直接打开这个地址

## 适合 3 到 20 人试用

- 一台固定开机的 Windows 电脑就够
- 文档统一保存在这台电脑的 `data/documents`
- 账号统一保存在这台电脑的 `data/users.json`

## 你需要注意

- 这台电脑要保持开机
- Windows 防火墙如果拦截 `3100` 端口，需要放行 Node.js 或该端口
- 同事必须和主机在同一个局域网

## 更稳一点的做法

- 不要用个人电脑，改用一台前台机、文控机或小型服务器
- 给这台机器固定内网 IP
- 再让大家长期访问固定地址，例如：

```text
http://192.168.1.23:3100
```

## 默认账号

- `admin / Admin@123`
- `editor / Editor@123`
- `viewer / Viewer@123`

## 试用完成后建议

- 把 `data/documents` 做每日备份
- 再考虑上正式内网服务器、域名、HTTPS、审批流和版本历史
